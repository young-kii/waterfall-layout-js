## Installing

### Package manager

Using npm:

```bash
npm install waterfall-layout-js
```

### CDN

Using jsDelivr CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/waterfall-layout-js@latest/dist/index.min.js"></script>
```

Using unpkg CDN:
```html
<script src="https://unpkg.com/waterfall-layout-js@latest/dist/index.min.js"></script>
```
